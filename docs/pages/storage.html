快速上手
StorageManager · 注册适配器 · 读写文件
v
import photon.storage // 1. 创建 StorageManager mut manager := storage.new_manager() // 2. 注册存储适配器 manager.register('local', storage.new_local_adapter('/var/uploads')) manager.register('s3', storage.new_s3_adapter('my-bucket', 'us-east-1')) // 3. 读写文件 mut disk := manager.get('local') // 写入文件 disk.write('avatars/alice.png', avatar_bytes, .public)! // 读取文件 content := disk.read('avatars/alice.png')! // 删除文件 disk.delete('avatars/alice.png')! // 检查文件是否存在 exists := disk.exists('avatars/alice.png') // 获取文件元数据 meta := disk.metadata('avatars/alice.png')! println(meta.size) // 文件大小 println(meta.mime_type) // MIME 类型 println(meta.visibility) // .public_ / .private_ // Laravel 等价：Storage::disk('local')->put('path', $content) // Spring 等价：ResourceLoader / S3Template

Storage 接口
实现自定义适配器
v
pub interface Storage { mut: write(path string, content []u8, visibility Visibility) ! delete(path string) ! set_visibility(path string, visibility Visibility) ! pub: read(path string) ![]u8 exists(path string) bool metadata(path string) !FileMetadata size(path string) !i64 last_modified(path string) !i64 visibility(path string) !Visibility list(path string) ![]string copy(from string, to string) ! move(from string, to string) ! } // 可见性控制 pub enum Visibility { public_ // 公开可访问（CDN 可读） private_ // 私有（需签名 URL） }

S3 适配器
AWS / MinIO / 兼容存储
v
// 创建 S3 适配器 mut s3 := storage.new_s3_adapter('my-bucket', 'us-east-1') s3.set_endpoint('https://s3.amazonaws.com') // 或 MinIO: http://localhost:9000 s3.set_credentials(access_key, secret_key) manager.register('s3', s3) // 上传文件到 S3 manager.get('s3').write('uploads/report.pdf', pdf_bytes, .private)! // 生成签名 URL（临时访问私有文件） url := s3.signed_url('uploads/report.pdf', ttl_minutes: 15)!
🚀 生产实践
  • 开发环境使用 Local 适配器，生产环境使用 S3
  • 上传文件使用 .private 可见性，通过签名 URL 按需授权
  • 使用 copy() 而非 read() + write() 在同存储间复制文件
  • 大文件使用分片上传（S3 multipart upload）
  • 文件路径使用 UUID 或哈希，避免路径冲突和信息泄露

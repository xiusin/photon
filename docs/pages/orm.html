实体定义
Touchable · Identifiable · 生命周期钩子

import photon.orm import time // 定义实体 — 嵌入 Touchable 自动管理时间戳 pub struct User { pub mut: id int [primary; sql: serial] name string [sql_type: 'TEXT']; string [sql_type: 'TEXT'; nonull] email string [sql_type: 'TEXT'; unique] age int is_active bool [sql_default: '1'] orm.Touchable // 自动管理 created_at / updated_at } // 生命周期钩子 pub fn (mut u User) before_insert() ! { // 插入前的校验/转换 if u.name.len == 0 { return error('name cannot be empty') } u.email = u.email.to_lower() } pub fn (mut u User) after_insert() ! { // 插入后：发事件、清缓存等 println('User ${u.id} created') } pub fn (mut u User) before_update() ! { // 更新前 } pub fn (mut u User) after_find() ! { // 查询后：解密敏感字段、计算派生属性等 }

import orm // V 原生 ORM import photon.orm // Photon 增强层 // 1. 初始化 OrmManager（多库路由） mut om := orm.new_orm_manager() om.register_connection('default', .sqlite, db)! om.register_connection('replica', .sqlite, replica_db)! // 2. 创建 Repository mut user_repo := orm.new_base_repository[User](om, 'default')! // 3. CRUD 操作 // 创建 mut user := User{name: 'Alice', email: 'alice@example.com', age: 30} user_repo.insert(mut user)! // 自动触发 before_insert → auto-touch → after_insert // 查询 found := user_repo.find_by_id(1)! // 按 ID 查询 all := user_repo.find_all()! // 全部查询 // 更新 mut existing := user_repo.find_by_id(1)! existing.name = 'Alice Updated' user_repo.update(mut existing)! // 自动更新 updated_at // 删除 user_repo.delete_by_id(1)! // 按 ID 删除 // Spring 等价：extends JpaRepository<User, Integer> // Laravel 等价：class UserRepository extends Eloquent

实体注解
JPA 风格注解 · 编译期实体映射

Photon ORM 提供 JPA 风格的注解系统，在编译期扫描实体元数据，实现声明式实体映射：

注解说明Spring JPA 等价
@[entity]标记 struct 为 JPA 实体@Entity
@[table('users')]指定表名@Table(name = "users")
@[column('user_name')]指定字段列名@Column(name = "user_name")
@[id]标记主键字段@Id
@[generated_value]自动生成主键@GeneratedValue
@[nullable]允许 NULL@Column(nullable = true)
@[length(255)]列长度@Column(length = 255)
@[precision(10)] / @[scale(2)]小数精度/标度@Column(precision=10, scale=2)
@[unique]唯一约束@Column(unique = true)
@[index('idx_name')]索引@Index(name = "idx_name")
@[transient]跳过持久化@Transient
@[enum_type('status_type')]枚举类型映射@Enumerated
@[embeddable] / @[embedded]嵌入值对象@Embeddable / @Embedded
@[version]乐观锁版本字段@Version
v
import photon.orm @[entity] @[table('users')] pub struct User { @[id] @[generated_value] id int @[column('user_name')] @[length(100)] @[nullable] name string @[column('email_address')] @[unique] email string @[column('age')] age int @[column('status')] @[enum_type('user_status')] status string @[version] version int @[transient] computed_field string } // scan_entity[T]() scans comptime annotations / 编译期扫描实体注解 meta := orm.scan_entity[User]() println(meta.table_name) // 'users' println(meta.columns.len) // number of mapped columns // validate[T]() validates entity constraints / 验证实体约束 result := orm.validate[User](user) if result.has_errors() { for err in result.errors { println('${err.field}: ${err.message}') } }

DTO 验证
ValidationError · ValidationResult · validate[T]

Photon ORM 提供结构化的数据验证机制，支持编译期约束扫描与运行时校验：

类型 / 方法说明Spring 等价
ValidationError单个验证错误（field, message, constraint）FieldError
ValidationResult验证错误集合BindingResult
validate[T](entity)验证 struct 字段约束@Valid + BindingResult
v
import photon.orm pub struct CreateUserDTO { pub: name string email string age int } // Validate DTO / 验证 DTO dto := CreateUserDTO{name: '', email: 'invalid', age: -1} result := orm.validate[CreateUserDTO](dto) if result.has_errors() { for err in result.errors { println('Field: ${err.field}, Error: ${err.message}, Constraint: ${err.constraint}') } } // Spring equivalent: @Valid + BindingResult // Laravel equivalent: FormRequest validation

Repository 增强
find_by_field · exists_by_field · count_by_field · paginate

BaseRepository[T] 提供增强查询方法，支持按字段查找、存在性检查、统计与分页：

方法说明Spring 等价
find_by_field(field, value)按字段值查找实体findByXxx()
exists_by_field(field, value)按字段检查是否存在existsByXxx()
count_by_field(field, value)按字段统计数量countByXxx()
paginate(page, page_size)分页查询findAll(Pageable)
PageResult[T]分页结果（含元数据）Page<T>
v
import photon.orm mut repo := orm.new_base_repository[User](om, 'default')! // Find by field / 按字段查找 users := repo.find_by_field('status', 'active')! // Check existence / 检查是否存在 exists := repo.exists_by_field('email', 'alice@example.com')! // Count by field / 按字段统计 count := repo.count_by_field('status', 'active')! // Pagination / 分页查询 page_result := repo.paginate(1, 20)! // page 1, 20 per page println('Total: ${page_result.total}, Pages: ${page_result.total_pages}') for user in page_result.items { println(user.name) } // Spring equivalent: PagingAndSortingRepository // Laravel equivalent: User::paginate(20)
🚀 生产实践
  • 始终嵌入 Touchable 让 Photon 自动管理 created_at / updated_at
  • 使用 before_insert 做数据校验，而非在 Controller 层做
  • 多库场景下用 OrmManager 路由：写操作走 master，读操作走 replica
  • 派生查询适合简单查询，复杂查询使用 OrmAdapter + 手写 SQL
  • 使用 @[entity] + @[table] + @[column] 注解实现编译期实体映射，避免手写 SQL
  • 使用 validate[T]() 在 Service 层做数据校验，而非在 Controller 层
  • 使用 paginate() 实现标准分页，返回 PageResult[T] 包含完整分页元数据

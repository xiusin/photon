Tracer · Span · TraceContext

import photon.tracing // 1. 创建 Tracer mut tracer := tracing.new_in_memory_tracer() // 2. 开启根 Span mut span := tracer.start_span('http.request') span.set_attribute('http.method', 'GET') span.set_attribute('http.url', '/api/users') // 3. 创建子 Span（自动关联父 Span） mut child := tracer.start_child_span(span, 'db.query') child.set_attribute('db.statement', 'SELECT * FROM users') child.finish() // 标记完成 // 4. 结束根 Span span.finish() // 5. 查询所有 Span spans := tracer.list_spans() println(spans.len) // 2

线程安全设计
RwMutex + 独立 Span Mutex + arc4random
组件
线程安全策略
InMemoryTracer
RwMutex：start/finish 用写锁，list_spans 用读锁
Span
每个 Span 独立 Mutex，不同 Span 并发无竞争
ID 生成
C.arc4random()（线程安全 CSPRNG），非 V 的 rand
时间戳
time.ticks() + C.time(0)，惰性转 time.Time

📧
快速上手
SMTP 配置 · 发送邮件
v
import photon.mailer // 1. 配置 SMTP smtp_config := mailer.SmtpConfig{ host: 'smtp.example.com' port: 587 username: 'noreply@example.com' password: 'app-password' encryption: .starttls } // 2. 创建 Mailer mut m := mailer.new_mailer(smtp_config) // 3. 使用 Fluent Builder 构建邮件 email := mailer.new_email() .from('noreply@example.com', 'MyApp') .to('alice@example.com', 'Alice') .cc('bob@example.com') .subject('Welcome to MyApp!') .html_body('<h1>Welcome, Alice!</h1><p>Your account is ready.</p>') .text_body('Welcome, Alice! Your account is ready.') .reply_to('support@example.com') // 4. 发送 m.send(email)! // Spring 等价：JavaMailSender / MimeMessage // Laravel 等价：Mail::to('alice@example.com')->send(new WelcomeMail())

添加文件附件
v
// 添加附件 file_content := os.read_file('invoice.pdf')! attachment := mailer.new_attachment( 'invoice.pdf', file_content.bytes(), 'application/pdf' ) email := mailer.new_email() .to('customer@example.com') .subject('Your Invoice') .html_body('Please find your invoice attached.') .attach(attachment) m.send(email)!

模板渲染
变量替换 · 复用模板
v
// 模板文件：templates/welcome.html // <h1>Welcome, {{name}}!</h1> // <p>Your account at {{app_name}} is ready.</p> mut renderer := mailer.new_template_renderer('templates') html := renderer.render('welcome.html', { 'name': 'Alice' 'app_name': 'MyApp' })! email := mailer.new_email() .to('alice@example.com') .subject('Welcome!') .html_body(html) // Laravel 等价：Mail::send('emails.welcome', $data)

Log 传输
开发环境 · 不实际发送
v
// 开发环境使用 Log Transport（邮件写入日志，不实际发送） mut m := mailer.new_log_mailer() m.send(email)! // 输出到日志：[MAIL] To: alice@example.com, Subject: Welcome! // Laravel 等价：MAIL_MAILER=log
🚀 生产实践
  • 使用 SMTP 的 STARTTLS 或 TLS 加密，避免明文传输
  • 邮件发送失败使用 Queue 模块异步重试
  • 批量发送使用 mailer.send_batch()，避免逐封发送
  • 开发环境使用 Log Transport，避免误发真实邮件
  • HTML 邮件同时提供 text_body，兼容不支持 HTML 的客户端

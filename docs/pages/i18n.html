快速上手
加载资源包 · 解析消息 · 占位符替换
v
import photon.i18n // 1. 创建 MessageSource mut ms := i18n.new_resource_bundle_message_source('messages', 'config/i18n') // 2. 加载语言包 ms.load_locale(.en)! ms.load_locale(.zh)! // 3. 解析消息（支持 {0} {1} 占位符） msg := ms.resolve('greeting', .zh, '张三') // → "你好,张三" msg2 := ms.resolve('welcome', .en, 'Alice', 'MyApp') // → "Welcome to MyApp, Alice!" // Spring 等价：MessageSource.getMessage("greeting", args, Locale.CHINA)

资源包格式
TOML 文件命名 · 占位符语法

资源包文件使用 TOML 格式，命名规则为 {basename}_{lang}.toml：

toml
# config/i18n/messages_zh.toml greeting = "你好,{0}" welcome = "欢迎来到{1},{0}!" error.not_found = "资源未找到" error.unauthorized = "未授权访问"
toml
# config/i18n/messages_en.toml greeting = "Hello, {0}" welcome = "Welcome to {1}, {0}!" error.not_found = "Resource not found" error.unauthorized = "Unauthorized access"

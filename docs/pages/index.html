Photon Framework

面向商业化应用的 V 语言企业级框架，对标 Spring Boot。编译期依赖注入、注解驱动、零运行时反射，极致性能与极低心智成本的完美平衡。

🚀 编译期 DI ⚡ 零反射开销 📦 注解驱动 🔒 类型安全 🌐 中间件生态
⚡
快速上手
5 分钟构建你的第一个 Photon 应用

1. 安装

bash
# 安装 V 语言（需要 0.4.x+） git clone https://github.com/vlang/v cd v && make # 安装 Photon 框架 v install photon # 或从源码安装 git clone https://github.com/your-org/photon.git cd photon && v self .

2. 最小应用

v
module main import photon.core import photon.web // 定义一个 Service @[component] pub struct HelloService { @[autowired] config &ConfigService } pub fn (s &HelloService) greet(name string) string { return 'Hello, ${name}! Config: ${s.config.get("app.name")}' } // 定义一个 Controller @[controller] pub struct AppController { web.BaseController veb.Context @[autowired] hello_service &HelloService } @[get('/hello/:name')] pub fn (mut app AppController) hello(name string) veb.Result { return app.ok('{"message":"${app.hello_service.greet(name)}"}') } // 启动应用 fn main() { mut app := core.new_application_context() app.set_profiles(['dev']) app.register(core.BeanDefinition{ type_name: 'HelloService' scope: .singleton }) app.refresh()! // 启动 Web 服务器 mut server := web.new_server(app) server.run(8080) }

3. 运行

bash
v run main.v # 🚀 Photon server started on http://localhost:8080 curl http://localhost:8080/hello/World # {"message":"Hello, World! Config: MyApp"}

📜
设计原则
编译期优于运行期 · 显式优于隐式 · 约定优于配置

⚙️ 编译期 DI

所有 @[autowired] 依赖在编译期通过 comptime $for 扫描解析，零运行时反射开销。启动即巅峰性能。

🔒 类型安全

V 语言的强类型系统 + 编译期检查 = 循环依赖检测、类型不匹配在编译期报错，而非运行时崩溃。

📦 注解驱动

@[component]、@[get: '/path']、@[cacheable] — 声明式编程，极低心智成本。

🔌 可插拔架构

模块间通过 trait 解耦，按需引入。不用 Cache？不 import 即可，零开销。

🔄 向后兼容

公开 API 变更必须经过弃用周期，遵循语义化版本控制（SemVer）。

🏭 生产就绪

SQL 注入防护、XSS 转义、CSRF Token、敏感信息脱敏 — 安全不是可选项。

快速上手
MeterRegistry · Counter · Gauge · Timer
v
import photon.metrics // 1. 创建注册中心 mut registry := metrics.new_meter_registry() // 2. Counter — 单调递增计数器（请求数、字节数） mut counter := registry.counter('http.requests.total') counter.increment() counter.increment_by(5) println(counter.value()) // 6 // 3. Gauge — 瞬时值（队列深度、温度） mut gauge := registry.gauge('queue.depth') gauge.set(42) gauge.increment() gauge.decrement() // 4. Timer — 耗时统计（请求数 + 总耗时） mut timer := registry.timer('http.request.duration') timer.record(150 * time.millisecond) timer.record(230 * time.millisecond) println(timer.count()) // 2 println(timer.total_time()) // 380ms

// 导出 Prometheus 文本格式 (v0.0.4) output := registry.format_prometheus() // # TYPE http_requests_total counter // http_requests_total 6 // # TYPE queue_depth gauge // queue_depth 42 // # TYPE http_request_duration timer // http_request_duration_count 2 // http_request_duration_total 0.380 // 配合 Web 模块暴露 /metrics @['/metrics'; get] pub fn (mut app App) metrics() veb.Result { return app.text(app.registry.format_prometheus()) }

线程安全
细粒度锁 · 无全局竞争

Metrics 模块采用细粒度锁策略，避免全局锁竞争：

层级
锁策略
Registry
RwMutex：查找用读锁，创建用写锁（双检锁）
Meter (Counter/Gauge/Timer)
每个 Meter 独立 Mutex，不同 Meter 不互斥
